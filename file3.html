CONTENT A

Streaming refers to any type of multimedia file that can be easily streamed (or viewed) without having to download the file first. There are two types of streaming; regular streaming and live streaming.

○ Regular streaming pertains to any type of multimedia file that is already recorded and later on posted on the internet.
○ Live streaming is a type of streaming that is happening in real-time.
